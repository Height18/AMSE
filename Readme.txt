Bienvenue sur notre projet d'application de soustraction de background en Flutter et Flask.

Il est à noter que l'api n'a pas été implémentée par nous, il s'agit d'une API basé sur u2-net disponible sur internet.

Pour lancer le projet, il faut tout d'abords veiller à ce que le téléphone et l'ordinateur soient connectés au même réseau.

Désactiver le pare-feu de l'ordinateur si nécessaire pour autoriser la connexion vers l'API depuis le téléphone.

Enfin, renseigner son adresse ipv4 dans la variable uri (ligne 45 du fichier main.dart).

Pour lancer la partie back en flask, il suffit d'ouvrir le projet avec un IDE pour python (type Pycharm), naviguer vers le dossier Flask Back (cd flask back), installer les dépendances requises (pip install -r requirements.txt) puis lancer l'application (python app.py).

Pour la partie flutter, il suffit de compiler le projet pour voir l'application sur son téléphone.
